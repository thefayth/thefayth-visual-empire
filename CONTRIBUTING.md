# Contributing

This is a protected public project surface, not an open-source codebase.

Public discussion may focus on documentation clarity, accessibility, typos, and non-sensitive project explanation. Contributions must not include private source code, credentials, copied third-party assets, sensitive records, customer data, private screenshots, or operational instructions.

Pull requests that attempt to reconstruct the private engine, add implementation details, weaken ownership language, or expose sensitive material should be rejected.
